# LoveMoney

#Frontend: 
+ Flutter: 2.10.3
+ Dart: 2.16.1
+ DevTools: 2.9.2

#Backend:
+ Nodejs
+ Framework: express
+ MongoDb

#UI/UX:
+ Figma

#To run frontend: 
+ if you use emulator, change enviroment to local: Enviroment.LOCAL
+ if you use real device, change enviroment to real device: Enviroment.REAL_DEVICE (your real device and your computer ip address is same)

#To run backend: 
+ npm i
+ npm run dev
